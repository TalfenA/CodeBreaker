# Java Code Breaker

An object oriented piece of Java code to break three different cyphers

## Description

This code makes use of object oriented programming principles to read text files, perform string manipulation according to provided parameters, and produce a saveable string outcome.

## Getting Started

### Dependencies

* JDK 19 or later

### Installing

* It is recommended to run the application in IntelliJ 2022 or later
* All text files should be located on the same level as the src, out, and .idea folders
* ceasar-key contains a number which is the amount to shift the alphabet by
* keyed-ceasar-key contains a number to shift the alphabet by and the key word to encrypt with
* vigenere-key contains the key to use with vigenere cypher

### Executing program

* The entire folder including text files should be loaded into IntelliJ as a project
* Run the application from the "Application" class in src
* Select the cypher to use
* Check the currently loaded key with 'Show key' and edit if necessary
* Load the text file to encrypt or decrypt. Include file extensions.
      This can be the provided useText.txt or your own text file
* Select whether to encrypt or decrypt
* When encrypting, you can save output to a new text file.

* The supplied example 'useText.txt' code is encrypted using Keyed Caesar Cypher
